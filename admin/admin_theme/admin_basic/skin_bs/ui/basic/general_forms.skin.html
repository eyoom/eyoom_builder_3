Eyoom form

이윰빌더에서 사용하는 시즌3 Eyoom form의 ShortCodes 입니다.

소스코드는 해당 예제에 마우스 포인트를 갖다대면 버튼이 나타나며, 클릭하면 소스코드를 볼 수 있습니다.

Eyoom Form 적용을 위하여 소스코드는 class="eyoom-form" 으로 감싸주어야 합니다.


<!--form 태그의 경우-->
<form action="#" class="eyoom-form">
	소스코드
</form>

<!--div 태그의 경우-->
<div class="eyoom-form">
	소스코드
</div>
